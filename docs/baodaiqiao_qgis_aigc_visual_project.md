# 宝带桥社区志 QGIS AIGC 可视化项目

## 项目目标

把《宝带桥社区志》中的地名、时间、空间和里程碑意义转化为可交互、可统计、可 3D 演示的 QGIS 项目。

核心问题：

- 哪些地名在志书中出现最多？
- 为什么这些地名高频？
- 它们在高德地图/卫星图上的位置在哪里？
- 能否用柱形图、3D 沙盘和动画镜头表达社区发展？

## AIGC 工作流

1. **志书文本提取**
   - PDF 提取为全文文本。
   - 从全文中提取地名、年份、上下文。

2. **地名统计与解释**
   - 统计精选地名出现频次。
   - 生成 `为什么高频` 的解释字段。
   - 关联主题：遗产文化、基层治理、开发建设、交通水系、农业工业、教育民生。

3. **空间定位**
   - 已定位地名生成地图点。
   - 高频缺坐标地名生成待补点表。
   - 使用高德纯地图/高德卫星图作为底图。

4. **QGIS 互动可视化**
   - 高频地名互动标注层。
   - 高频地名柱形统计层。
   - 20 秒里程碑时间层。
   - 3D 沙盘和动画关键帧导引。

5. **AIGC 叙事输出**
   - 从统计结果生成解说词。
   - 从 3D Map View 导出动画帧。
   - 后续可接入剪辑、配音、字幕和图文报告。

## 推荐图层组合

### 统计互动视图

打开：

- `高德纯地图_AIGC底图`
- `宝带桥社区志_地名提及统计互动层`
- `高频地名_统计标注_可点击`

用途：

- 点击地名点查看提及次数。
- 查看为什么高频。
- 查看坐标质量。

### 高频柱形视图

打开：

- `高德卫星_AIGC底图`
- `宝带桥社区志_高频地名柱形统计`
- `高频地名_提及次数柱形图_3857`
- `高频地名_柱顶数值标签_Top10`

用途：

- 柱高表达地名在志书中的提及频次。
- 标签显示排名、地名、次数、柱高。
- 点击柱体查看解释字段。

### 3D 沙盘视图

打开：

- `宝带桥社区志_3D沙盘实践`
- `3D_地名块体_面`
- `3D_里程碑意义光柱_面`
- `3D_里程碑时间路径_线`

用途：

- 在 `视图 > 新建3D地图视图` 中查看 3D 沙盘。
- 用光柱表达社区发展里程碑。

### 动画关键帧视图

打开：

- `宝带桥社区志_3D动画关键帧导引`
- `动画关键帧目标点_K0-K6`
- `动画镜头目标路径_20秒`

用途：

- 在 3D Map View 中按 K0-K6 设置动画 keyframe。
- 导出 20 秒动画帧。
