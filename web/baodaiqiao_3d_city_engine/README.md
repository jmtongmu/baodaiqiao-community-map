# 宝带桥历史里程碑3D城市版

这是 `宝带桥历史里程碑3d游戏引擎展示` 分支上的第一版可行性原型，简称 **3d城市版**。

它和现有 `baodaiqiao_aigc_map` 地图版不同：地图版以 MapLibre/二维底图为主，3d城市版以 Babylon.js 3D 游戏引擎场景为主，把社区志中已经整理出的地名、提及频次、坐标和里程碑事件投射到一个可旋转、可缩放、可点击的 3D 沙盘中。

## 启动

双击：

```text
一键启动_宝带桥3D城市版.bat
```

或在本目录运行：

```powershell
python -m http.server 8383 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:8383/
```

## 当前验证点

- 使用 Babylon.js 本地引擎文件，不依赖运行时外网。
- 复用现有 `mention_columns.geojson`、`milestones.geojson`、`stats.json`。
- 用局部投影把经纬度转换为 3D 坐标，保持宝带桥社区内地物的相对空间关系。
- 使用生成式航拍城市底纹作为地面基板，再叠加程序化道路、水系、桥梁和建筑块。
- 采用 Simple 3D Buildings 思路生成现代建筑层：住宅塔楼、板式住宅、商业玻璃楼、学校/社区服务建筑、滨水建筑，并预留后续接入 QGIS/OSM 建筑轮廓的空间。
- 用柱体高度表达地名提及频次。
- 用发光光柱表达社区志里带地点的里程碑事件。
- 支持点击柱体/光柱查看地名统计、里程碑意义和志书原文。
- 支持时间轴播放，按年份逐步显隐里程碑。

## 后续方向

1. 把 QGIS 校核后的边界、道路、水系导出为更准确的 3D 底座。
2. 将高德/卫星图层作为参考纹理或手工建模底图，而不是继续依赖传统地图控件。
3. 为 20 秒里程碑动画增加相机关键帧、镜头路径和导出录制流程。
4. 继续比较 Babylon.js、Three.js、PlayCanvas、CesiumJS 的适配度。
